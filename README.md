## Disclaimer

Thoses are my personal dotfiles, they works fine for me on a Lenovo T450s laptop runing debian Buster and a QHD dual monitor, I did not tests them on any other configuration. 

## Install

```bash
git clone https://github.com/nicolasbeauvais/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Dependencies

- [Debian](https://www.debian.org/index.html) 10 (Buster / Testing)
- [I3 Gaps](https://github.com/Airblader/i3)
- [I3 Blocks](https://github.com/vivien/i3blocks)
- [Z shell](https://github.com/zsh-users/zsh) with a custom
  [Prezto](https://github.com/nicolasbeauvais/prezto)
- [Dunst](https://github.com/dunst-project/dunst)
- [Doom Emacs](https://github.com/hlissner/doom-emacs)
- [Rofi](https://github.com/DaveDavenport/rofi)
- [RClone](https://github.com/ncw/rclone)
- [Redshift](https://github.com/jonls/redshift)
